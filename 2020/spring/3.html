Operators in C++

Wondering what an operator is?

An operator is a symbol that tells the compiler to perform specific mathematical or logical manipulations. We have a lot of built-in operators in C++. An operand is a data or variable on which the operation is to be performed on.

The following are some types of operators in C++:

  • Arithmetic operators.
  • Relational operators.
  • Logical operators
  • Bitwise operators.
  • Assignment operators.
  • Special operators
  • Conditional operators

Modulus operator and returns the remainder of an after-integer division.

int NUM2 = 4

++

Increment operator increases integer value by one.

int NUM2 = 4
int Total = NUM1 + NUM2

–

Decrement operator decreases an integer by one.

int NUM1 = 5
int NUM2 = 4
